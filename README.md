# Theorakl

**The signs are speaking.**

A mystical decision-making app that helps users interpret the signs the universe sends them.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Tech Stack

- Next.js 14
- TypeScript
- Vercel (hosting)
- Supabase (coming soon)
- Claude API (coming soon)

## Deploy

Push to GitHub, connect to Vercel, done.
